##Usage:

	$ git clone --depth 1 https://github.com/encorehu/repo.git new-project-name
	$ cd new-project-name
	$ git create new-project-name

explain:

	--depth <depth>
	Create a `shallow` clone with a history truncated to the specified number of revisions.
